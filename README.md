# Atelier Lumière — Landing premium

## Vídeos

En `src/App.jsx` puedes activar y configurar tus 2 vídeos:

- `heroVideoEnabled` + `heroVideoSrc` para la portada.
- `parallaxVideoEnabled` + `parallaxVideoSrc` para el bloque parallax.

Por defecto quedan en `false` para evitar errores si aún no has subido los MP4.
